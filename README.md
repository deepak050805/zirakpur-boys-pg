# Zirakpur Boys PG Website

A modern and responsive website developed for a client to showcase their Boys PG accommodation in Zirakpur. The website is designed to provide a clean user experience and highlight essential information such as rooms, facilities, pricing, and location.

## 🚀 Features
- Fully responsive design (mobile-friendly)
- Interactive image gallery with lightbox view
- Room and pricing sections
- Integrated Google Maps location
- One-click Call and WhatsApp contact options
- Smooth scrolling and animations

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📍 Purpose
This project was built as a client-based solution to help increase online visibility and provide potential customers with a seamless way to explore and contact the PG.
